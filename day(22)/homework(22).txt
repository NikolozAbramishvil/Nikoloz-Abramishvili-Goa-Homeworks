Github-არის ვერსიების კონტოლის მარტივი საშუალება. 
git init-მისი დაწერის შემთხვევაში ჩვენ ვეუბნებით კომპიუტერს რომ ვიყენებთ გითს.
Git add .-ამით ჩვენ ვამამტებთ ახალ ვერსიაში ფაილებს. 
Git commit -m "name"-ამით ჩვენ ვქმნით ახალ ვერსიას და ბრჭყალებში ვწერთ ამ ვერსიის სახელს. 
Git status -ამით ჩვენ ვანხულობთ თუ რა ფაილები ემატება ახალ ვერსიაში.
Git reset-მაის დახმარებით ჩვენ ვარესეტებთ ანუ ის ფაილები რაც უნდა დამატებულიყო ახალ ვერსიაში წაიშლება და არაფერი არ დაემატება. 
Git log-ამის მეშვეობით მე შემიძლია ვნახო ის ვერსიები რომლების შევქმენი.
